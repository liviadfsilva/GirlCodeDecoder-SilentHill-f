## About üéå
Girl Code Decoder is a command-line application created to decode the numerical cipher known as 'girl code' from the game Silent Hill f, used by many students as their locker combinations.

## Running the Application üíª

1. **Clone the repository:**

   ```sh
   git clone https://github.com/liviadfsilva/GirlCodeDecoder-SilentHill-f.git
   cd GirlCodeDecoder-SilentHill-f
   ```

2. **Build the Docker image:**
   
   ```bash
    docker build -t girl-code-decoder .
   ```

3. **Run the application:**
   
   ```bash
    docker run -it girl-code-decoder
   ```
<br/>

## Puzzle Location üß©
You'll find this puzzle once you reach the school building. Make sure you explore all rooms and hallways so you don't miss any of the necessary documents that will contain the letters that will help you decipher the locker combinations.
